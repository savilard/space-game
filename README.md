# Space game

We fly in space, we surf the Universe

## Requirements to the environment
* Linux
* Python 3.10+

## How to install
Download the repository content:
```bash
git clone https://github.com/savilard/space-game.git
```
Go to the game folder:
```bash
cd space-game
```

## How to run
```bash
python app/main.py
```
